# 💰 PayCurve - Simulateur de Paie Variable & Commissions

Ce projet est un outil automatisé doté d'une interface graphique (Front-End) permettant de calculer instantanément le **Taux d'Atteinte (Achievement Rate)** en fonction d'un **Taux de Performance (TR)** saisi. 

Il intègre nativement 4 barèmes complexes de commissions basés sur des règles de paliers (linéaires, fixes et quadratiques).

## 🚀 Fonctionnalités
- **4 profils gérés** : Standard Curve, Manager Curve 2025, Courbe Manager IS 2025, Inside Sales Curve 2025.
- **Formules dynamiques** : Gestion automatique des paliers inclusifs/exclusifs, des multiplicateurs au carré (`TR * TR`) et des plafonds/planchers fixes.
- **Interface temps réel** : Saisie des performances via un formulaire web simple et mise à jour immédiate du résultat.
